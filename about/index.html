头像

  • 这张照片是在 2015 年 9 月份骑行拉萨的时候拍的。那天我们小队休整，大伙儿提议去塔公草原玩，到达目的地之后远远地可以看见雅拉雪山，很美。后来不知怎样的想法，我跟其中一个队友决定走走看能不能到雪山脚下，于是我们俩按照直线向雪山奔去。期间经过了不大不小的沼泽、数不清的小山，捡到了一对牦牛角，还看见了一个女人骑着马从山脚跑到了山顶。大概走了四个小时后，离雪山还是很远。在一个山顶上，我远眺着雅拉，坐在草地上，用自拍杆拍下了这张照片。

龙岩 → 日喀则

  • 时间：201508 - 201510
  • 原定：终点是尼泊尔的加德满都，后因地震重建关闭了陆路口岸，导致最后的结点落在了日喀则。
  • 路线图：本次骑行路线大致可以分成三个部分：第一部分，从福建龙岩到四川成都，走的是 G319；第二部分，从四川成都到西藏拉萨，走的是 G318；第三部分，从西藏拉萨到日喀则，走的是 G318。龙岩 → 长汀 → 瑞金 → 萍乡 → 株洲 → 长沙 → 常德 → 吉首 → 凤凰 → 松桃 → 涪陵 → 重庆 → 成都 → 雅安 → 泸定 → 新都桥 → 理塘 → 巴塘 → 芒康 → 左贡 → 邦达 → 然乌 → 波密 → 通麦/排龙 → 鲁朗 → 八一 → 工布江达 → 墨竹工卡 → 拉萨 → 浪卡子 → 江孜 → 日喀则。

南宁 → 吉隆坡

  • 时间：201712 - 201802
  • 路线图：南宁 → 凭祥 → 谅山 → 河内 → 府里 → 清化 → 荣市 → 承天顺化 → 岘港 → 会安 → 绥和 → 芽庄 → 大叻 → 美奈 → 胡志明 → 金边 → 暹粒 → 芭提雅 → 曼谷 → 华欣 → 童颂 → 合艾 → 亚罗士打 → 槟城 → 美罗 → 吉隆坡。
